Full‑Stack Einkaufsliste (React, TypeScript, Express, MongoDB)

Projekt Information

  • Kategorie: Software/App
  • Klient: BlickSolution GmbH, Idstein
  • Projekt Datum: 13. Januar 2026
  • Projekt URL: GitHub

Dieses Projekt ist eine vollständige Full‑Stack‑Webanwendung, mit der Nutzer Produkte zu einer Einkaufsliste hinzufügen, abhaken und löschen können. Das Backend basiert auf Express + TypeScript und stellt eine REST‑API bereit, während das Frontend mit React + TypeScript eine moderne, responsive Benutzeroberfläche bietet. Die Daten werden in MongoDB gespeichert und über Mongoose verwaltet. Der Fokus lag auf sauberer Architektur, Typsicherheit, klarer Komponentenstruktur und einer einfachen, aber ansprechenden UI.

Folgendes ist hier zum Einsatz gekommen:

Bereich Technologien
Frontend React, TypeScript, Material UI, Fetch API
Backend Node.js, Express, TypeScript, Mongoose
DB MongoDB
Tools Git, dotenv, ts-node-dev, npm, VS Code
